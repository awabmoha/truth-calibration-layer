The experiment folder contains early TCL-v0 diagnostics only: frozen LLM hidden states -> small probe -> correctness/confidence score. These diagnostics are preliminary and do not validate full TCL. The Python virtual environment should be excluded from GitHub because it is large and can be recreated from requirements.txt.
